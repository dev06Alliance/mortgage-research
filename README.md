# mortgage-research

This workspace now includes a simple documentation website for the SMSF project.

The documentation site is available under the `docs/` folder for GitHub Pages.

Open `docs/index.html` in a browser to preview the site locally. Available pages:

- `step-1-solution.html`
- `step-2-security.html`
- `step-3-smsf-details.html`
- `step-4-income.html`
- `step-5-financial-position.html`
- `step-6-loan-summary.html`

Step 3 is built from `step-3_who-on-the-load.md`; Step 1 and Step 2 are currently placeholders.

## GitHub Pages Deployment

1. In your repository settings on GitHub, go to **Pages**.
2. Set the source branch to `main` or `master` and the folder to `docs`.
3. Save the settings.

After a few minutes, your site will be available at `https://<username>.github.io/<repository>/`.
